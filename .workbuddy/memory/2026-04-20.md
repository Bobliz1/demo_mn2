## 下午修改

### 完成的功能

1. **benchmark.py 新增内容**：
   - 新增 `FUNCTION_MAXIMA` 字典，为各函数定义理论最大值
   - 新增 4 个测试函数：
     - `multipeak_func`：多峰函数（理论最大值 5.0）
     - `flat_region_func`：平坦区域+尖峰（理论最大值 3.0）
     - `asymmetric_func`：非对称函数（理论最大值 10.0）
     - `noisy_func`：高噪声函数（理论最大值 4.0）
   - 新增 `BENCHMARK_FUNCTIONS` 函数注册表
   - 新增 `get_function_max()` 辅助函数
   - 修改 `ExperimentLogger`：新增基线BO vs MN-BO对比表格输出

2. **main.py 新增功能**：
   - 修改 `run_inner_bo()` 支持传入目标函数
   - 新增 `run_baseline_bo()` 运行无变换基线BO
   - 修改 `main()`：同时运行基线BO和MN-BO，输出对比结果
   - 新增对比可视化：左图基线BO，右图MN-BO
   - 新增 `run_all_functions()` 批量运行所有测试函数
   - 支持命令行参数：`python main.py multipeak_func` 或 `python main.py --all`

### 使用方法

```bash
python main.py                    # 运行默认 deceptive_trap_func
python main.py multipeak_func     # 运行指定函数
python main.py --all              # 运行所有测试函数
```
