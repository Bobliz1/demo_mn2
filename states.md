### 备忘录：如何在命令行中手动运行实验

如果你希望自己手动在 Mac 终端中运行实验，请按照以下流程操作：

**1. 进入项目工作区**
```bash
cd /Users/linus/demo_mn2
```

**2. 激活虚拟环境**
```bash
conda activate bo_env
```
*(如果终端提示找不到 conda 命令，请先执行 `source ~/.zshrc`)*

**3. 运行实验命令**
```bash
# 进入交互式菜单
python src/main.py

# 跑全部函数的完整实验
python src/main.py --all

# 跑单函数的快速验证（推荐在修 Bug 期间使用）
python src/main.py cliff_func -R 1 -M 5 -N 8 -I 8
```
